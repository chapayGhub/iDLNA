#DLNA

DLNA 标准DMS,DMC,DMR在IOS上的实现。  

##项目简介

本项目使用PlatinumKit开源库,作为dlna调用接口。  

使用cocoapods 管理依赖库，应该使用 xcworkspace工程文件，而不是xcodeproj.

##Refrence
  
本项目的开发主要依照upnp及dlna标准。文档来自dlna及upnp官网。
